Register a cloud point in order to maximize the given score image.

Given an array of points this function tries to apply affine transformations in order to better fit these points maximizing the score image. The score image can be computed as the Inverted Euclidean Distance Transform as in iftEuclideanScoreImage(), or any other maximization score.
